### Prerequisites

-Ruby: 2.6.3 Rails: 5.2.3

### Install and Setup

-Clone repo to your local
> git clone < address >

-Install gems with:
> bundle install

-Setup database with:
> rails db:create

> rails db:migrate

### Usage

-Start server with:
> rails server

Open http://localhost:3000/ in your browser.

### Run tests
-Run:
> rpsec --format documentation
